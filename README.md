# Time-Manager-CPP 🚀

这是一个由 C++ 驱动的时间管理工具。

## 👥 团队分工
- **后端 (组长)**: 负责 C++ 逻辑实现、数据持久化。
- **前端 (组员)**: 负责 UI 界面设计、数据可视化。

## 🛠️ 技术架构
- **核心引擎**: C++ 11
- **数据存储**: `tasks.txt` (UTF-8 编码)
- **前端界面**: HTML5 / JavaScript / CSS3

## 📂 接口约定 (给前端同学看)
后端会将任务实时写入项目根目录下的 `tasks.txt`。
- **文件格式**: 每行一条任务字符串。
- **更新频率**: 实时追加。

## 🚀 如何运行
1. 编译并运行 `main.cpp` 开始记录任务。
2. 双击 `web/index.html` 在浏览器中查看漂亮的任务界面。
